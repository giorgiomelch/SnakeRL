# Snake Game - Solving with Reinforcement Learning
This project explores various Reinforcement Learning (RL) techniques to solve the classic Snake game. The main goal is to train agents to maximize their score by learning optimal policies for navigating the game board and collecting food while avoiding collisions.

For a comprehensive explanation of the approaches, training methods, and results, please refer to https://lilac-hip-934.notion.site/e5b72f87dcad4f109620c4662f8f9dad?v=10dfdf11b644801dbe4c000c9a005657&pvs=74. This site provides a detailed breakdown of each method, its performance, and the challenges faced during training.
